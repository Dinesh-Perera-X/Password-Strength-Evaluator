# Password Strength Evaluator

A simple Python command-line utility built to assess the mathematical strength and complexity of a user's password.

## 🚀 Features Implemented
* **Day 1:** Baseline character-length constraint validation (Minimum 8 characters).
* **Day 2:** Iterative character-loop to verify numerical digits.

## 🛠️ How To Run
Ensure you have Python installed on your Linux system, then execute:
```bash
python3 password_check.py
